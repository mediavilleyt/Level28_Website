Showcase

This is the page where we will explain different AI's and what they are specifically used for.

Github Copilot

Github Copilot is an open source software that helps you to create code. It automatically generates what it thinks you want to type next. You'll need a student account or a subscription to use it in your code editor.

ChatGPT

ChatGPT is an open source software, a language model made by OpenAI. Its purpose is to assist users with text-based prompts, generating suitable responses. It's free and can be used by people of any age; you just need to create a free account.

DALL-E

DALL-E is an open source software that specializes in image generation, also made by OpenAI. There are different versions, including DALL-E 2 and DALL-E 3. DALL-E 2 is currently free, while DALL-E requires payment.
