this is the page where we will explain different AI's and what they are specifically used for

Github Copilot

Github copilot is an open source software that helps you to create code, it will automatically generate what it thinks that you want to type next, you'll need a student account or a subscription to be able to use it in your code editor

ChatGPT

ChatGPT is an open source software that is a language model made by OpenAI, it's purpose is to assist users with text based prompts, it will automatically generate a response that is suitable with the user's question or text. It's a free software that can easily be used for people any age, you just need to make a free account and you're all set!
